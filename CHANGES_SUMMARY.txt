================================================================================
WORKFLOW - QUY TRÌNH MỚI
================================================================================

CÁ NHÂN (ADMIN):
  1. Chọn "Đã giao" từ select status
  2. BẮTBUỘC upload hình ảnh chứng minh
  3. Thêm ghi chú (tùy chọn)
  4. Click cập nhật
  → Hình ảnh lưu + lịch sử cập nhật

NGƯỜI DÙNG (USER):
  1. Nhìn thấy nút "Xác nhận đã nhận hàng" khi status='delivered'
  2. Click nút → xác nhận dialog
  3. Hệ thống cập nhật status → 'received'
  4. Thời gian lưu vào user_received_at
  → Đơn hàng hoàn tất

================================================================================
TRẠNG THÁI ĐƠN HÀNG (STATE MACHINE)
================================================================================

TRƯỚC:
  pending → confirmed → processing → shipping → delivered (THE END)

SAU:
  pending → confirmed → processing → shipping → delivered → received (THE END)
                                                    ↓
                                              (Admin xác nhận giao)
                                              (Có hình ảnh chứng minh)
                                                    ↓
                                              (User xác nhận nhận)
                                              (Đơn hàng hoàn tất)
